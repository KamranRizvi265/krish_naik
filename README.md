# Krish Naik - LangChain Learning & AI Model Integration

A comprehensive learning repository exploring LangChain, AI model integration, and advanced LLM concepts through Jupyter notebooks.

## 📚 Overview

This repository contains a series of educational Jupyter notebooks that progressively cover:
- Introduction to LangChain fundamentals
- Integration with various AI models
- Tool usage and function calling
- Message handling and chat patterns
- Structured output generation

The project demonstrates practical applications of modern language models using LangChain, with support for multiple LLM providers including OpenAI, Google Generative AI, Vertex AI, and Groq.
